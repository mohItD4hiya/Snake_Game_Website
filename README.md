## Snake Game Web App

A classic Snake Game built using web technologies with enhanced gameplay features for a better user experience.

#### Features 
-  High Score Tracking – Keeps record of your best performance
-  Level System – Progress through levels as your score increases
-  Dynamic Speed – Game speed increases with higher levels for added challenge
-  Pause & Resume – Take a break anytime without losing progress
-  Restart Option – Quickly restart the game and try again
-  Smooth Controls & Gameplay
#### Tech Stack
- HTML
- CSS
- JavaScript

#### Overview

This project is a fun implementation of the classic Snake Game with modern features like level progression and speed scaling. It’s designed to improve user engagement while demonstrating core JavaScript game logic and DOM manipulation.
